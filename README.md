# MakerVerse

Projeto de Extensão - IFPE Belo Jardim.

## 🛠 Como Colaborar

1. Use o **Google Chrome** ou **Microsoft Edge**.
2. Clone o repositório: 
   `git clone https://github.com/liviaisadora27/Makeverse_Projeto.git`
3. No Construct 3, vá em **Menu > Projeto > Abrir > Pasta de projeto**.

## 🚀 Comandos Básicos

- Antes de iniciar: `git pull`
- Após salvar: `git add .`, `git commit -m "o que foi feito"`, `git push`
